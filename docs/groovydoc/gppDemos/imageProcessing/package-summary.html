Package gppDemos.imageProcessing

Package gppDemos.imageProcessing Description

The image processing package demonstartes the used of the ImageEngine, which itself comprises an ImageRoot and a number of ImageNode processes whereby the image processing is partitioned over the nodes. The demonstartion show how a pipeline fo such ImageEngine components can be constructed to undertake a sequence of kernel operations on an image.

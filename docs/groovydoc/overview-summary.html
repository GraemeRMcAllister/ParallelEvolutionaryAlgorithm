gppDemos  
gppDemos.MCpi The demos in this package use some of the Farm type patterns in the package gppLibrary.patterns to implement a solution to the calculation of pi using a MonteCarlo method.
gppDemos.QuickSort This package implements the QuickSort algorithm.
gppDemos.concordance Concordance provides a typical problem scenario that justifies the use of a parallel solution, both a sequential and parallel version are provided that use the same data classes.
gppDemos.goldbach.data Data classes used in the Primes and the Goldbach Conjecture solution
gppDemos.goldbach.scripts Primes and the Goldbach Conjecture presents a solution to the problem of creating prime numbers and then using those primes to validate the Goldbach conjecture.
gppDemos.imageProcessing The image processing package demonstartes the used of the ImageEngine, which itself comprises an ImageRoot and a number of ImageNode processes whereby the image processing is partitioned over the nodes.
gppDemos.jacobi The Jacobi demo finds the solution to the simultaneous equations represented by Ax = b, where A is the nxn matrix of coefficients, x represents the n unkowns for which the value is required and b is n values that result from the vector sum of Ax.
gppDemos.mandelbrot.cluster The cluster version of Mandelbrot causes the calculation to be distributed over a number of nodes and a host connected by a TCP/IP network.
gppDemos.mandelbrot.data Data Classes used in the Mandelbrot solution.
gppDemos.mandelbrot.scripts Mandelbrot - a solution to creating the Mandelbrot set.
gppDemos.mapReduceV0 This demonmstration implements a Map Reduce architecture to count the number of words in an input text file.
